    Julia Set Fractal Generator (2018)

    Details

    • Generates Julia Set from a given startpoint
    • Here's part of the code that does the fractal calculation by pixel: 
                                      
im = PIL.Image.new('RGB', (int(500*size),int(500*size)), 'white')
pix = im.load()
for y in range(0,int(math.floor(500*size))):
    for x in range (0, int(math.floor(500*size))):
        zx = xscaled(x)
        zy = yscaled(y)

        iteration = 0
        max_iteration = 1000
    
        while (zx*zx + zy*zy < 6400  and  iteration < max_iteration):
            xtemp = zx*zx - zy*zy
            zy = 2*zx*zy + float(y_entry.get()) #Siegel Disk Fractal -0.390541-0.586788i
            zx = xtemp + float(x_entry.get())
            iteration = iteration + 1
    
        if (iteration == max_iteration):
            pix[x,y] = (0,0,0)

        root.update_idletasks()
        root.update()
        
    upscaled = im.resize((1000,1000))
    upscaled_display = ImageTk.PhotoImage(upscaled)
    image_id = canvas.create_image((500,500), image=upscaled_display)
    Motivation For Project

    My motivation for creating this project was because I surprised to learn that Julia set fractals were created by iterating complex numbers through a very simple mathermatical equation: f(z) = z^2 + c. It was surprisingly easy to create a user-interactive program that generates Julia sets from a user-provided startpoint and outputs an image.
